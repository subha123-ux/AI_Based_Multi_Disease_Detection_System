How System Works

The system accepts a medical image, preprocesses it, analyzes features using a GNN deep learning model, predicts disease probability, displays real-time results, and generates a downloadable report while maintaining prediction history.

Step 1: Upload a medical image such as X-ray, MRI, CT Scan, or skin image using the secure upload system.

Step 2: The uploaded image is preprocessed automatically. The system resizes, normalizes, and enhances image quality for accurate analysis.

Step 3: A trained Graph Neural Network (GNN) model analyzes the image by extracting medical features and patterns.

Step 4: The Artificial Intelligence model compares detected patterns with trained disease datasets to identify possible abnormalities.

Step 5: The system predicts the disease along with confidence score based on model probability.

Step 6: Real-time prediction results are displayed instantly on the screen for quick diagnosis support.

Step 7: Patient details and prediction history are securely stored at Admin side for future reference.

Step 8: Users can download an AI-generated medical report for consultation with healthcare professionals.
